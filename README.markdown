# Simple deployer example
    This application provides a simple automated deploy with mina to rails

## Go!

1. Clone the source
2. go to project path (`cd ~path~/celtab-take-a-picture/`)
3. run de blundler (`bundle install`)
4. run de bower (`bower install`)
5. to run the app (`rails s`)
6. Then open [http://localhost:3000/](http://localhost:3000/)
